# InfoSync-AI-Web-Scraper
AI-powered news platform that intelligently scrapes multiple sources, aggregates  content, and synthesizes comprehensive summaries. Includes real-time web scraping,  LLM integration for smart summarization, professional audio narration, search  history tracking, and a modern UI. Deploy locally with no external API keys needed.
